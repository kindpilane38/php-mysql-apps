# Contact Management System

A PHP–MySQL based web application that allows users to submit contact messages,
which are securely stored in a database and managed through an admin dashboard.

This project focuses on backend fundamentals such as database connectivity,
server-side validation, and structured data handling.

---

## Features

- Contact form with server-side input validation
- Secure data storage using MySQL
- Admin dashboard to view all contact messages
- Messages ordered by timestamp for better readability
- CRUD operations implemented using Core PHP
- Clean and simple backend flow

---

## Tech Stack

- PHP (Core PHP)
- MySQL
- HTML5
- CSS3
- Apache (XAMPP / Localhost)

---

## How It Works

1. Users submit contact details through the contact form.
2. Data is validated and sanitized on the server side.
3. Valid records are stored in the MySQL database.
4. Admin can view all messages in the dashboard.
5. Messages are displayed in descending order of timestamp.
